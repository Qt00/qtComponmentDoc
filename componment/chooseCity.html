城市选择 ​

提供按城市和按省份选择所有的城市，支持搜索和跳转

按需引入 ​

js
//main.ts
import chooseCity from 'qt-component/chooseCity'
import 'qt-component/chooseCity/style.css'

app.use(chooseCity)

基本用法 ​

具体演示见城市选择器

代码示例 ​

js
    <my-choose-city></my-choose-city>

API ​

事件 ​

事件名说明参数
changeCity按城市选择时触发val:City 城市数据
changeProvince按省份选择时触发val:string 城市名称

City数据格式 ​

key说明类型
id城市编码number
spell城市拼音string
name城市名字string
